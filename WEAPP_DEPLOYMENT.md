# 📱 微信小程序完整部署指南

## 🎯 部署流程概览

```
1. 注册小程序
   ↓
2. 配置项目
   ↓
3. 编译代码
   ↓
4. 上传代码
   ↓
5. 配置服务器域名
   ↓
6. 提交审核
   ↓
7. 发布上线
```

---

## 📝 第 1 步：注册小程序

### 1.1 访问微信公众平台

打开浏览器访问：
```
https://mp.weixin.qq.com/
```

### 1.2 注册账号

1. 点击右上角 **"立即注册"**
2. 选择 **"小程序"**
3. 填写账号信息：
   - **邮箱**：未注册过微信的邮箱
   - **密码**：设置登录密码
   - **确认密码**：再次输入密码

### 1.3 选择主体类型

选择适合您的主体类型：

#### 个人认证
- ✅ 免费
- ✅ 注册快速
- ✅ 可以使用基础功能
- ❌ 某些功能受限（支付、高级 API）

**适用场景**：
- 个人开发者
- 学习和测试
- 小型应用

#### 企业认证
- ✅ 功能完整
- ✅ 支付功能
- ✅ 高级 API
- ⚠️ 需要营业执照
- ⚠️ 需要 300 元/年 认证费

**适用场景**：
- 商业应用
- 需要支付功能
- 企业级应用

### 1.4 完成注册

1. 扫描二维码绑定微信
2. 填写个人信息（或企业信息）
3. 完成邮箱验证
4. 登录后台

---

## 📝 第 2 步：获取 AppID

### 2.1 查看 AppID

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入 **"设置"** → **"基本设置"**
3. 在 **"账号信息"** 中找到 **"AppID"**
4. 复制 AppID（格式：`wx1234567890abcdef`）

### 2.2 记录重要信息

记下以下信息，后续需要使用：
- **AppID**：小程序的唯一标识
- **AppSecret**：API 调用密钥（开发设置中）
- **登录邮箱**：用于登录后台
- **登录密码**：用于登录后台

---

## 📝 第 3 步：配置项目

### 3.1 修改项目配置

编辑 `project.config.json`：

```json
{
  "miniprogramRoot": "./dist",
  "projectname": "word-learning-app",
  "description": "单词学习助手",
  "appid": "wx1234567890abcdef",  // ← 替换为你的 AppID
  "setting": {
    "urlCheck": false,  // 开发阶段设为 false
    "es6": true,
    "enhance": true,
    "compileHotReLoad": true,
    "postcss": true,
    "minified": true
  },
  "compileType": "miniprogram"
}
```

**重要**：
- 将 `"appid"` 替换为您的小程序 AppID
- `urlCheck` 设为 `false`（开发阶段）

### 3.2 在 Coze 环境中修改

如果您在 Coze 环境中，可以使用修复脚本：

```bash
cd /workspace/projects
./fix-appid.sh
```

---

## 📝 第 4 步：编译小程序

### 4.1 本地编译

在项目根目录运行：

```bash
pnpm build:weapp
```

**输出**：
```
✅ 编译成功
输出目录：dist/
```

### 4.2 检查编译结果

确认以下文件存在：
```bash
ls -la dist/
```

应该看到：
- `app.js`
- `app.json`
- `app.wxss`
- `pages/` 文件夹

---

## 📝 第 5 步：使用微信开发者工具

### 5.1 下载微信开发者工具

访问下载页面：
```
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
```

选择适合您操作系统的版本：
- Windows 64位
- macOS

### 5.2 登录

1. 打开微信开发者工具
2. 扫描二维码登录（使用注册小程序的微信账号）

### 5.3 导入项目

1. 点击 **"+"** 或 **"导入项目"**
2. 选择项目目录：
   - 点击 **"选择"**
   - 选择项目的 `dist` 文件夹
3. 输入 AppID：
   - 选择 **"小程序"**
   - 输入您的 AppID（如：`wx1234567890abcdef`）
4. 输入项目名称：
   - `单词学习助手`
5. 点击 **"导入"**

### 5.4 测试功能

1. **预览功能**：
   - 点击 **"预览"** 按钮
   - 扫描二维码
   - 在手机微信中打开

2. **测试功能**：
   - 输入单词：`hello`
   - 点击 **"开始分析"**
   - 查看结果
   - 点击 **"朗读单词"**

---

## 📝 第 6 步：配置服务器域名

### 6.1 准备后端服务

确保后端服务已部署并可公网访问：

**测试后端**：
```bash
curl https://your-backend-url.railway.app/api/health
```

应该返回：
```json
{
  "status": "success",
  "data": "2026-03-02T14:30:00.000Z"
}
```

### 6.2 添加合法域名

#### 在微信公众平台配置：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入 **"开发"** → **"开发管理"** → **"开发设置"**
3. 找到 **"服务器域名"** 部分
4. 点击 **"修改"**
5. 在 **"request 合法域名"** 中添加：
   ```
   https://your-backend-url.railway.app
   ```
6. 点击 **"保存并提交"**

**重要**：
- ✅ 必须使用 HTTPS
- ✅ 必须是完整域名（不能是 IP）
- ✅ 不能带端口号
- ✅ 域名需要备案（中国大陆）

### 6.3 开发阶段临时配置

在开发测试阶段：

1. 打开微信开发者工具
2. 点击右上角 **"详情"**
3. 在 **"本地设置"** 标签中
4. 勾选 **"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"**
5. 重新编译

**注意**：
- ⚠️ 这是临时方案
- ⚠️ 发布前必须配置合法域名
- ⚠️ 否则无法通过审核

---

## 📝 第 7 步：上传代码

### 7.1 修改配置

在发布前，修改配置：

```json
{
  "setting": {
    "urlCheck": true  // ← 发布前设为 true
  }
}
```

### 7.2 重新编译

```bash
pnpm build:weapp
```

### 7.3 上传代码

在微信开发者工具中：

1. 点击右上角 **"上传"** 按钮
2. 填写版本号：
   - 版本号：`1.0.0`
3. 填写项目备注：
   - 示例：`初始版本，单词学习功能上线`
4. 点击 **"上传"**

等待上传完成，会提示 **"上传成功"**。

---

## 📝 第 8 步：提交审核

### 8.1 登录微信公众平台

访问：
```
https://mp.weixin.qq.com/
```

### 8.2 进入版本管理

1. 进入 **"管理"** → **"版本管理"**
2. 点击 **"开发版本"** 标签
3. 找到刚上传的版本

### 8.3 提交审核

1. 点击 **"提交审核"** 按钮
2. 填写审核信息：

#### 基本信息
- **页面路径**：`pages/index/index`
- **功能页面截图**：上传截图（2-5 张）
- **测试账号**：提供测试账号（如需要）

#### 隐私设置
- **用户隐私保护指引**：填写隐私政策
- **是否收集用户信息**：选择 **"否"**

#### 功能描述
填写功能说明：
```
单词学习助手是一款帮助用户学习英语单词的小程序。
主要功能：
1. 单词智能分析：输入单词，获取词性、音标、中文意思和例句
2. 语音朗读：支持单词和例句的语音播放
3. 简洁界面：清晰的页面设计，易于使用

适用人群：英语学习者、学生、英语爱好者
```

3. 点击 **"提交"**

### 8.4 等待审核

**审核时间**：
- 个人认证：1-3 个工作日
- 企业认证：1-7 个工作日

**审核状态查询**：
1. 登录微信公众平台
2. 进入 **"管理"** → **"版本管理"**
3. 查看审核状态

---

## 📝 第 9 步：发布上线

### 9.1 审核通过后

1. 登录微信公众平台
2. 进入 **"管理"** → **"版本管理"**
3. 找到审核通过的版本
4. 点击 **"发布"** 按钮

### 9.2 填写发布信息

- **版本描述**：
  ```
  v1.0.0 - 初始版本发布

  新功能：
  - 单词智能分析
  - 语音朗读
  - 简洁界面

  修复：
  - 无
  ```

3. 点击 **"确认发布"**

### 9.3 发布成功

发布成功后，小程序即可在微信中搜索到。

## 🔍 版本管理

### 版本类型

| 类型 | 说明 | 访问方式 |
|------|------|----------|
| **开发版本** | 开发中，正在测试 | 微信开发者工具 |
| **体验版本** | 内测版本 | 扫描体验二维码 |
| **审核版本** | 正在审核中 | 不可访问 |
| **线上版本** | 已发布，用户可用 | 微信搜索 |

### 版本切换

在微信开发者工具中：
1. 点击右上角 **"版本"**
2. 选择要切换的版本
3. 点击 **"切换"**

## 📊 常见问题

### Q1: 审核被拒绝怎么办？

**常见原因**：
- 功能描述不清晰
- 页面截图不完整
- 隐私政策未填写
- 合法域名未配置

**解决方法**：
1. 查看拒绝原因
2. 修改问题
3. 重新上传代码
4. 重新提交审核

### Q2: 如何加快审核速度？

**方法**：
- 确保功能描述清晰
- 提供完整的截图
- 填写详细的隐私政策
- 个人认证审核更快

### Q3: 开发阶段如何测试？

**方法**：
1. 使用体验版
2. 扫描预览二维码
3. 添加体验成员
4. 使用真机调试

### Q4: 后端服务如何部署？

**方法**：
1. 使用 Railway 部署后端
2. 配置 HTTPS 域名
3. 添加合法域名
4. 确保服务稳定运行
