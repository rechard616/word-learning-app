# 🚀 微信小程序快速部署

## 📋 快速开始（5 分钟）

### 1. 注册小程序
访问：https://mp.weixin.qq.com/
- 点击"立即注册"
- 选择"小程序"
- 填写信息完成注册

### 2. 获取 AppID
1. 登录微信公众平台
2. 进入"设置" → "基本设置"
3. 复制 AppID（格式：wx1234567890abcdef）

### 3. 修改配置
编辑 `project.config.json`：
```json
{
  "appid": "你的AppID"
}
```

### 4. 编译代码
```bash
pnpm build:weapp
```

### 5. 导入项目
1. 打开微信开发者工具
2. 导入 `dist` 文件夹
3. 使用 AppID 导入

### 6. 上传发布
1. 点击"预览"测试
2. 点击"上传"代码
3. 登录微信公众平台提交审核
4. 审核通过后发布

---

## 🔧 使用部署脚本

```bash
# 快速部署准备
./deploy-weapp.sh
```

脚本会自动：
- 检查 AppID 配置
- 检查 urlCheck 配置
- 编译小程序代码
- 验证编译结果
